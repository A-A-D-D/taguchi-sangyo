# WBS

| 大項目 | 中項目 | 内容 |
|---|---|---|
| 要件定義 | 業務整理 | 現行のFAX・メール・電話運用を整理 |
| 要件定義 | 権限整理 | 全体管理者、本部責任者、現場責任者、現場施工者、発注先の権限定義 |
| 要件定義 | 注文区分整理 | 一般注文、買掛注文、現場発注の区分整理 |
| 基本設計 | DB設計 | 会員、会社、現場、商品、価格、注文、納品、原価の設計 |
| 基本設計 | 画面設計 | フロント、マイページ、管理画面、発注先画面の設計 |
| 基本設計 | ステータス設計 | 注文、承認、納品、請求の状態遷移を定義 |
| 詳細設計 | 価格改定 | 価格履歴、適用開始日、注文時単価固定の設計 |
| 詳細設計 | 原価集計 | 現場別・月別・商品別の集計設計 |
| 詳細設計 | CSV | 商品、価格、注文、原価集計の入出力設計 |
| UI制作 | EC画面 | 商品一覧、商品詳細、カート、注文確認 |
| UI制作 | 管理画面 | ダッシュボード、注文管理、商品管理、価格管理 |
| UI制作 | 現場画面 | 現場別注文、承認待ち、注文履歴 |
| 開発 | 認証・権限 | ログイン、ロール、アクセス制御 |
| 開発 | 会員管理 | 会員登録、登録情報変更、会社情報管理 |
| 開発 | 現場管理 | 現場登録、連絡先、車輌規制、施工者紐付け |
| 開発 | 商品管理 | 商品、カテゴリ、画像、販売可否 |
| 開発 | 価格管理 | 通常価格、買掛価格、仕入価格、価格履歴 |
| 開発 | 注文機能 | 一般注文、買掛注文、カート、注文履歴 |
| 開発 | 承認機能 | 現場施工者依頼、現場責任者承認、本部承認 |
| 開発 | 納品管理 | 納期回答、出荷、納品完了、分納 |
| 開発 | 原価管理 | 現場別発注量、工事原価、月次集計 |
| 開発 | 帳票 | 注文伝票、現場別集計表、CSV出力 |
| テスト | 単体テスト | 機能単位の確認 |
| テスト | 結合テスト | 注文から納品・原価反映までの確認 |
| テスト | 権限テスト | ロールごとの閲覧・操作制限確認 |
| テスト | 運用テスト | 本部・現場・発注先の実務想定確認 |
| 移行 | マスタ投入 | 商品、価格、会員、現場データ投入 |
| リリース | 本番公開 | 環境設定、公開、初期サポート |
