# 現状課題

## 発注業務の分散

現在の発注業務は、
- FAX
- メール
- 電話
- 紙管理

が混在しており、情報の一元化が困難です。

## 現場別原価の把握負荷

現場ごとの発注量・原価を把握するために、
複数資料の確認や集計作業が必要となっています。

## 価格改定管理

建材価格は日々変動するため、

- 古い価格での発注
- 価格反映漏れ
- 見積との差異

が発生しやすい状態です。

## 属人化

担当者依存により、

- 履歴追跡
- 納期確認
- 発注確認

が特定担当者に集中しています。

## 情報共有不足

本部・現場・発注先間で、
リアルタイムな情報共有が難しい状態です。
