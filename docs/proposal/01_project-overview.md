# プロジェクト概要

## システム名称

田口産業 建材発注・原価管理システム

## システム概要

本システムは、

- 建材EC
- 現場別発注管理
- 承認フロー
- 原価管理
- 発注履歴管理

を統合した、
建材業向けWeb発注・業務管理システムです。

## 対象業務

- 一般注文
- 買掛注文
- 現場別発注
- 承認管理
- 納品確認
- 原価集計
- CSV出力

## 基本方針

- Webベース運用
- スマートフォン対応
- クラウド運用
- 段階導入
- 将来拡張可能設計
