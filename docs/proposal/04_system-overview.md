# システム全体像

## 主な機能

- 建材EC
- 会員管理
- 現場管理
- 商品管理
- 価格履歴管理
- 一般注文
- 買掛注文
- 承認フロー
- 原価集計
- CSV出力
- 管理ダッシュボード

## 想定利用者

| 区分 | 主な役割 |
|---|---|
| 全体管理者 | システム全体管理 |
| 本部工事部責任者 | 原価・承認管理 |
| 現場責任者 | 現場管理・発注 |
| 現場施工者 | 発注依頼 |
| 発注先 | 納期回答・出荷登録 |

## 基本方針

- 段階導入
- Webベース
- スマートフォン対応
- クラウド運用
- 将来拡張可能設計
