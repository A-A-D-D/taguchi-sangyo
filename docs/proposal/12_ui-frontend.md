# フロント画面一覧

| 画面 | 内容 |
|---|---|
| トップ | 一般注文、買掛注文への導線 |
| ログイン | IDまたはメールアドレス、パスワードでログイン |
| 新規会員登録 | 会社名、業種、住所、担当者、電話、FAX、メール、パスワード登録 |
| 商品一覧 | カテゴリ別の商品表示 |
| 商品詳細 | 商品画像、商品説明、サイズ、単価、数量、備考 |
| カート | 商品内容、数量、小計、送料、合計 |
| 注文情報入力 | 現場選択、お届け希望日、受取希望時間帯 |
| 注文内容確認 | お客様情報、現場情報、商品内容、支払い情報 |
| 注文完了 | 注文受付完了メッセージ |
