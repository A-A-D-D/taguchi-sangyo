# 管理画面一覧

| 画面 | 内容 |
|---|---|
| 管理者ログイン | 管理者認証 |
| ダッシュボード | 注文数、未承認、未納品、月次発注額 |
| 管理者情報 | 管理者名、受信メールアドレス、パスワード変更 |
| 会員一覧 | 会員検索、会社情報、連絡先、担当者確認 |
| 会員詳細 | 会員情報、所属現場、注文履歴 |
| 商品管理 | 商品登録、編集、非表示、カテゴリ設定 |
| 価格管理 | 通常価格、買掛価格、仕入価格、価格履歴 |
| 注文情報 | 注文検索、注文詳細、伝票確認 |
| 承認管理 | 承認待ち、差戻し、承認履歴 |
| 納品管理 | 納期回答、出荷状況、納品完了 |
| 原価管理 | 現場別・月別・商品別の原価集計 |
| CSV入出力 | 商品、価格、注文、原価データの入出力 |
| メールマガジン | タイトル、本文、送信先条件を指定して配信 |
